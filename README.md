## How to Build

### Development

#### Requirements

- [Node.js](https://nodejs.org/)
- [Volta](https://volta.sh/)

And install dependencies with bellow command.

```bash
npm install
```

#### Run on Dev Server

```bash
npm run build
```

### Production Build

```bash
npm run build
```
